## ğŸš€ **Getting Started**

Follow these simple steps to get the platform up and running locally:

### 1. **Clone the Repository**

```bash
git clone https://github.com/coderanirban/mindful-learning.git
cd mindful-learning
```

### 2. **Install Dependencies**

